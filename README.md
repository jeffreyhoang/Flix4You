# Flix4You

Flix4You is a **full-stack movie streaming website** built with **Django** and **React**, where users can create accounts and watch classic films online.  
The platform is hosted with **Render** (frontend) and **Railway** (backend), combining modern web technologies with nostalgic entertainment.

---

## Website

**Live Site:** [https://flix4you.onrender.com](https://flix4you.onrender.com)

---

## Features

- **User Authentication** – Create an account, log in, and manage your profile.  
- **Movie Streaming** – Watch a collection of old movies directly on the site.  
- **Search & Browse** – Find films by title, genre, or release year.  
- **Interactions** – Save, like, and comment on your favorite movies. 
- **Responsive UI** – Works seamlessly across desktop, tablet, and mobile.  

---

## Tech Stack

### **Frontend**
- React.js  
- React Router  
- Axios  
- HTML/CSS  

### **Backend**
- Django  
- Django REST Framework (DRF)  
- PostgreSQL (via Railway)  
- JWT Authentication  

### **Deployment**
- Frontend → **Render**  
- Backend → **Railway**  
- Database → **PostgreSQL**
